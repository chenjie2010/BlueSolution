三、安装说明

1. 客户端要求
   
（1） Windows 7 SP1 及以上版本

（2）.Net Framework 4.8（只需要安装一次）

3. 客户端安装步骤:
   
（1） 安装.Net Framework 4.8 （只需要安装一次）

（2） 安装客户端软件

4. 服务器端要求
   
（1） Windows Sever 2012 及以上版本

（2）.Net Framework 4.8（只需要安装一次）

5. 服务器端安装步骤
   
（1）安装.Net Framework 4.8（只需要安装一次）

（2）安装数据库

（3）创建七个数据库：Blue, Blue_1, Blue_2, Blue_3, Blue_4, Blue_5, Blue_Business, 并在Blue上执行SQL文件(发布的压缩包中已提供)

（4）安装服务器端软件

（5）在“服务器端管理界面”配置数据库设置（每次重新安装需要配一次）

（6）在“服务器端管理界面”启动服务（每次重新安装后需要启动一次）

（7）服务器端口：8090
